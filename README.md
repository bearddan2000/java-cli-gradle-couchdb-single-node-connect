# java-cli-gradle-couchdb-single-node-connect

## Description
A java gradle build, that connects to couchdb database.

## Tech stack
- java
- couchdb
- gradle
  - log4j
  - couchdb

## Docker stack
- couchdb:3.1
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
